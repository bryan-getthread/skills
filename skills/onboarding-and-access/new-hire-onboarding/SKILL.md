# New Hire Onboarding

**When to use:** An employee onboarding request has come in and needs the full provisioning checklist.

**What you get:** A completed or tracked onboarding: account, licenses, group access, hardware, and enforced MFA, with anything pending flagged.

## Steps

1. Gather full name, start date, role, department, manager, and required apps and groups.
2. Create the user account and mailbox.
3. Assign licenses and security groups for the role, and add to distribution lists, shared mailboxes, and Teams.
4. Prepare and record hardware.
5. Enforce MFA enrollment on first sign-in.
6. Summarize what was done and what is still pending, such as hardware shipment.

## Guardrails

- Base access on the role profile and the client's licensed solutions, not a generic default.
- Confirm before creating accounts or assigning licenses that carry cost.

## Consolidates

New hire onboarding, new user onboarding planner, and onboarding orchestrator skills.
